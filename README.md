# Aplicación vulnerable - Desarrollo Seguro

## Objetivo

Esta aplicación forma parte de un ejercicio práctico para la asignatura de Desarrollo Seguro de Software.

El objetivo es analizar una aplicación vulnerable, identificar problemas de seguridad y aplicar mejoras.

---

## Puesta en marcha

1. Instalar dependencias:

```
pip install flask
```

2. Inicializar la base de datos:

```
python init_db.py
```

3. Ejecutar la aplicación:

```
python app.py
```

4. Acceder a:

http://127.0.0.1:5000

## Tareas del ejercicio

- Analizar el código fuente
- Identificar vulnerabilidades
- Evaluar impacto
- Corregir los problemas detectados
- Documentar en el repositorio

## Nota

La aplicación contiene vulnerabilidades de forma intencionada para fines educativos.
